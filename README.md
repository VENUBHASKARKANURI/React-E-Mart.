 # 🛒 E-Mart – E-Commerce Web Application

E-Mart is a modern e-commerce web application built using React and Vite.

## 🚀 Features
- Product browsing (Mobiles, Computers, etc.)
- Smart search functionality
- Cart management using Context API
- Responsive UI design
- AI-based product recommendations

## 🛠️ Tech Stack
- React.js
- JavaScript
- HTML
- CSS
- Context API
- Vite
